### 2.1npm源没有指向npm.js

- 报错类型

![1642579012(1)](./1642579012(1).jpg)

- 解决方案

  ```
  * 全局安装 nrm
  	npm install nrm -g
  
  * 查看当前npm源
  	nrm ls
  
  * 切换源到npm.js
  	nrm use npm
  ```

### 2.2注册账号超时

- 报错类型

![1642579654(1)](./1642579654(1).jpg)



- 解决方案

  ```
  * 原因： 
  	初步判断： npm官网发送验证到我们指定邮箱的时间太久，甚至已经超过了有效期
  	最终确定： npm虽然的响应时间确实很久，但是请不要多次发送，窗口也不要关闭，一次发送对应一次验证就能注册成功
  ```
